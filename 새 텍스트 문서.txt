🧱 전체 개발 단계 (Step-by-Step)
🟩 1단계: 프로젝트 구조 및 MVVM 기본 세팅
📌 목표: 프로젝트 틀 만들고 화면 연결 구조 만들기

 WPF (.NET Core) 프로젝트 생성

 MVVM 구조 폴더 구성 (Model, ViewModel, View)

 MainWindow에서 메뉴 or 버튼으로 화면 이동 가능하게 구성

 RelayCommand / DelegateCommand 등 기본 커맨드 로직 작성

 필요한 ViewModel/Model 틀 미리 잡아놓기

🟨 2단계: 복지 프로그램 목록 불러오기 (CSV 기반)
📌 목표: CSV 데이터 → 프로그램 목록 UI 표시

 CSV 파일 불러오기 기능 구현

 ProgramModel 클래스 설계 (이름, 대상, 운영기관 등)

 ViewModel에서 CSV → ObservableCollection 변환

 View (XAML)에서 DataGrid로 목록 표시

 필터 기능 (지역, 대상 등)

🟦 3단계: 가상 사용자 정보 생성/관리
📌 목표: 성별, 나이, 지역 등 가진 사용자 정보 등록/보기

 UserModel 클래스 생성

 사용자 등록 화면 (입력 폼 + 저장 버튼)

 사용자 목록 화면 (DataGrid)

 ViewModel에서 사용자 리스트 관리

🟥 4단계: 복지 프로그램 추천 기능
📌 목표: 사용자 조건 기반으로 추천할 프로그램 선정

 사용자 선택 → 조건(성별, 나이, 지역) 추출

 조건에 맞는 프로그램 필터링

 추천 버튼 클릭 시 추천 내역 저장 (RecommendationModel)

 추천 결과를 사용자별로 저장

🟪 5단계: 참여 여부 기록 기능
📌 목표: 추천된 사용자에게 참여 여부 입력 가능하게 하기

 추천된 사용자 리스트에 참여 여부 (참여/불참 등) 선택 가능

 ViewModel에서 참여 상태 업데이트

 상태별로 색상/아이콘 등 시각적 표시

🟫 6단계: 통계 및 분석 화면
📌 목표: 참여율/인기 프로그램 등 시각적 분석 제공

 프로그램별 추천 수 / 참여 수 분석

 사용자 연령대, 지역별 참여율 분석

 인기 프로그램 순위 (가장 많이 추천 or 참여된 순)

 차트, 그래프 등을 사용한 시각화 (예: LiveCharts)

⚙️ 7단계: 부가 기능 및 마무리
📌 목표: 완성도 높은 앱 마무리

 추천 결과 내보내기 (엑셀 or PDF)

 데이터 저장 (CSV, JSON, or DB 선택 가능)

 예외 처리, 유효성 검사, 디자인 다듬기

 관리자 전용 설정 화면 (선택)



📌 1. 대시보드 (메인 화면)
간단한 요약 정보 (등록된 프로그램 수, 총 사용자 수, 참여율 등)

다른 화면으로 이동하는 메뉴 또는 버튼

✅ 역할: 앱의 홈 화면, 통계 요약, 네비게이션

📌 2. 복지 프로그램 목록 화면
CSV로부터 불러온 프로그램 리스트 표시

지역, 대상 등 필터 가능

프로그램 상세 정보 보기 (선택)

✅ 역할: 복지 프로그램 관리의 중심
✅ 개발 단계: 2단계 (CSV 불러오기)

📌 3. 가상 사용자 관리 화면
사용자 목록 (이름, 성별, 나이, 지역 등)

사용자 추가/수정/삭제

✅ 역할: 추천 대상인 사용자 가상 DB 관리
✅ 개발 단계: 3단계

📌 4. 프로그램 추천 화면
사용자 선택 → 조건 기반 프로그램 자동 추천

추천할 프로그램 리스트 확인 및 추천 실행 버튼

추천 결과 저장

✅ 역할: 관리자용 추천 실행 기능
✅ 개발 단계: 4단계

📌 5. 추천 결과 / 참여 관리 화면
추천된 사용자와 프로그램 리스트

참여 여부 체크 (참여 / 불참 / 미응답 등)

날짜별 필터, 상태별 필터 가능

✅ 역할: 추천 결과 확인 및 참여율 관리
✅ 개발 단계: 5단계

📌 6. 통계 및 분석 화면
프로그램별 추천/참여 현황

연령대/성별/지역별 참여율 시각화 (차트)

인기 프로그램 순위 표시

✅ 역할: 다음 운영을 위한 인사이트 제공
✅ 개발 단계: 6단계

📌 7. 설정 화면 (선택)
CSV 파일 경로 선택

기본 정보 설정

(DB 저장 시) DB 연결 정보 설정

✅ 역할: 데이터 로딩, 초기 설정 관리

📋 전체 화면 요약 표
화면 이름	주요 기능	개발 단계
📊 대시보드	요약 통계, 화면 이동	1단계
📂 프로그램 목록	CSV 데이터 불러오기, 목록 보기, 필터링	2단계
👤 사용자 관리	사용자 등록/수정/삭제	3단계
🎯 프로그램 추천	조건 기반 추천 실행	4단계
✅ 추천 결과/참여 관리	추천 내역 보기, 참여 여부 기록	5단계
📈 통계 분석	프로그램별 인기/참여율 차트	6단계
⚙️ 설정 화면	경로 설정, 초기 세팅	보조 기능
